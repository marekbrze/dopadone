# Converter Layer

## Overview

The Converter Layer transforms types between **DB types** (generated by sqlc) and **Domain types** (defined in internal/domain). It handles NULL values, type conversions, and ensures clean separation between layers.

**Key Characteristics**:
- **Bidirectional conversions**: DB ↔ Domain
- **NULL handling**: Converts sql.NullXXX types to domain types
- **Centralized**: All conversions in one place
- **Type-safe**: Maintains type safety across layers

## DB → Domain Conversions

### DbTaskToDomain

Converts database Task model to domain Task entity:

```go
// internal/converter/converter.go
func DbTaskToDomain(dbTask db.Task) domain.Task {
    status, _ := domain.ParseTaskStatus(dbTask.Status)
    priority, _ := domain.ParseTaskPriority(dbTask.Priority)
    
    var estimatedDuration domain.TaskDuration
    if dbTask.EstimatedDuration.Valid {
        estimatedDuration, _ = domain.ParseTaskDuration(int(dbTask.EstimatedDuration.Int64))
    }
    
    return domain.Task{
        ID:                dbTask.ID,
        ProjectID:         dbTask.ProjectID,
        Title:             dbTask.Title,
        Description:       nullStringToString(dbTask.Description),
        StartDate:         dbTask.StartDate,
        Deadline:          dbTask.Deadline,
        Priority:          priority,
        Context:           nullStringToString(dbTask.Context),
        EstimatedDuration: estimatedDuration,
        Status:            status,
        IsNext:            dbTask.IsNext == 1,
        CreatedAt:         dbTask.CreatedAt,
        UpdatedAt:         dbTask.UpdatedAt,
        DeletedAt:         dbTask.DeletedAt,
    }
}
```

## NULL Handling Patterns

### Null String → String

```go
func nullStringToString(ns sql.NullString) string {
    if ns.Valid {
        return ns.String
    }
    return ""
}
```

### Null String → *String (Pointer)

```go
func nullStringToPtr(ns sql.NullString) *string {
    if ns.Valid {
        return &ns.String
    }
    return nil
}
```

### String → Null String

```go
func stringToNullString(s string) sql.NullString {
    return sql.NullString{
        String: s,
        Valid:  s != "",
    }
}
```

**Usage in Service**:
```go
dbParams := db.CreateTaskParams{
    Title:       task.Title,
    Description: sql.NullString{String: task.Description, Valid: task.Description != ""},
    Context:     sql.NullString{String: task.Context, Valid: task.Context != ""},
}
```

## Domain → DB Conversions

When creating or updating records, services convert domain types to DB params:

```go
// In service layer
dbParams := db.CreateTaskParams{
    ID:                task.ID,
    ProjectID:         task.ProjectID,
    Title:             task.Title,
    Description:       sql.NullString{String: task.Description, Valid: task.Description != ""},
    StartDate:         task.StartDate,
    Deadline:          task.Deadline,
    Priority:          task.Priority.String(),
    Context:           sql.NullString{String: task.Context, Valid: task.Context != ""},
    EstimatedDuration: sql.NullInt64{Int64: int64(task.EstimatedDuration.Int()), Valid: task.EstimatedDuration != 0},
    Status:            task.Status.String(),
    IsNext:            boolToInt64(task.IsNext),
    CreatedAt:         task.CreatedAt,
    UpdatedAt:         task.UpdatedAt,
    DeletedAt:         task.DeletedAt,
}
```

**Note**: Domain → DB conversions are typically inline in service methods, not centralized in converter functions. This is because DB params structures vary by operation (Create vs Update).

## Converter Testing

```go
// internal/converter/converter_test.go
func TestDbTaskToDomain(t *testing.T) {
    now := time.Now()
    dbTask := db.Task{
        ID:          "task-123",
        ProjectID:   "proj-456",
        Title:       "Test Task",
        Description: sql.NullString{String: "Description", Valid: true},
        Status:      "todo",
        Priority:    "high",
        CreatedAt:   now,
        UpdatedAt:   now,
    }
    
    domainTask := DbTaskToDomain(dbTask)
    
    if domainTask.ID != "task-123" {
        t.Errorf("expected ID task-123, got %s", domainTask.ID)
    }
    
    if domainTask.Status != domain.TaskStatusTodo {
        t.Errorf("expected status todo, got %s", domainTask.Status)
    }
    
    if domainTask.Description != "Description" {
        t.Errorf("expected description 'Description', got %s", domainTask.Description)
    }
}

func TestDbTaskToDomain_NullDescription(t *testing.T) {
    dbTask := db.Task{
        ID:          "task-123",
        Title:       "Test Task",
        Description: sql.NullString{Valid: false}, // NULL
    }
    
    domainTask := DbTaskToDomain(dbTask)
    
    if domainTask.Description != "" {
        t.Errorf("expected empty description for NULL, got %s", domainTask.Description)
    }
}
```

## Best Practices

### 1. Always Handle NULL Values

❌ **Don't**:
```go
// Will panic if Description is NULL
description := dbTask.Description.String
```

✅ **Do**:
```go
description := nullStringToString(dbTask.Description)
```

### 2. Use Value Object Parse Methods

❌ **Don't**:
```go
status := domain.TaskStatus(dbTask.Status) // Unsafe, no validation
```

✅ **Do**:
```go
status, err := domain.ParseTaskStatus(dbTask.Status)
if err != nil {
    // Handle invalid status
}
```

### 3. Centralize NULL Handling

Use helper functions instead of repeating NULL checks:

```go
// ✅ Good: Reusable helper
func nullStringToString(ns sql.NullString) string {
    if ns.Valid {
        return ns.String
    }
    return ""
}

// ✅ Good: Use helper everywhere
description := nullStringToString(dbTask.Description)
context := nullStringToString(dbTask.Context)
```

### 4. Validate After Conversion

Converters don't fail; they return zero values for invalid inputs. Validate in domain factory instead:

```go
// Converter: Safe but may return invalid values
status, _ := domain.ParseTaskStatus("invalid") // Returns "", no error

// Domain Factory: Validates and rejects invalid values
task, err := domain.NewTask(params) // Returns error for invalid status
```

## Common Conversion Patterns

| From | To | Helper |
|------|----|----|
| `sql.NullString` | `string` | `nullStringToString()` |
| `sql.NullString` | `*string` | `nullStringToPtr()` |
| `sql.NullInt64` | `int` | Manual check + cast |
| `sql.NullTime` | `*time.Time` | Direct assignment |
| `string` | `sql.NullString` | `stringToNullString()` |
| Domain enum | `string` | `.String()` method |
| `string` | Domain enum | `domain.ParseXXX()` |
